This is a simple TDD project created for {Course} at {School} by {Author} in {Year}.

== Setup
[source,shell]
----
go mod tidy
----

== Run
[source,shell]
----
go run .
----

== Test

[source,shell]
----
// Run all tests
 go test -v ./...

// Run all tests with coverage
go test -cover -v ./...
----

== Coverage
[source,shell]
----
go test -coverprofile="coverage.out" ./...
go tool cover -html="coverage.out"
----
